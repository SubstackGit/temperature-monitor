# Dual-Scale Temperature Monitor for macOS

A beautiful SwiftUI Mac app displaying an animated dual-scale thermometer (°C/°F) with interactive preset buttons for everyday use.

## Features
- 🌡️ **Dual Scales**: Real-time Celsius and Fahrenheit readings side-by-side.
- 💧 **Animated Mercury**: Smooth spring animation as you adjust temperature.
- 🎯 **Interactive Presets**: Quick-jump to Common, Weather, Health, Cooking, and Aquarium temperatures.
- 📏 **Precise Scale**: Tick marks for every single degree (1°C).

## Requirements
- **OS**: macOS 13.0 (Ventura) or later
- **Xcode**: 15.0+

## How to Run
1. Clone or download this repository.
2. Open `Thermometer.xcodeproj` in Xcode.
3. Select your Mac as the destination.
4. Press `Cmd + R` to build and run.

## Preset Categories
| Category | Examples |
|----------|----------|
| **Common** | Room (20°C), Coffee (60°C), Hot Water (70°C) |
| **Weather** | Cold (10°C), Warm (30°C), Water (40°C) |
| **Health** | Cool Bath (25°C), Body Temp (37°C), Fever (40°C) |
| **Cooking** | Butter Soften (20°C), Eggs (45°C), Warm Oven (70°C) |
| **Aquarium** | Cold Water (15°C), Tropical (25°C), Too Warm (35°C) |
